# Прозрачный шлюз для анализа трафика ТВ-приставки Man-in-the-Middle (MITM Box)

[![Status: In Progress](https://img.shields.io/badge/status-in%20progress-yellow)]()

## 📋 О проекте

Этот проект практическое исследование сетевого стека Linux и протоколов передачи данных.
Системный блок в самодельном корпусе из оргстекла выступает в роли прозрачного шлюза (MITM)
между ТВ-приставкой на Android 11 и интернетом.

**Цель:** Перехватить и задокументировать абсолютно весь трафик приставки без использования
готовых скриптов. Каждый шаг выполняется и описывается вручную для глубокого понимания технологий.

## 🧰 Стек технологий

- **ОС:** Linux (дистрибутив будет указан позже)
- **Сетевые службы:** `hostapd`, `dnsmasq`
- **Межсетевой экран:** `nftables` / `iptables`
- **Анализ трафика:** `tcpdump`, `Wireshark`, `mitmproxy`
- **Документирование:** Markdown, Git, GitHub
- **Прочее:** (Будет указано позже)
